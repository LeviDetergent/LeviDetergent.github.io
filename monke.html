A website that was created using Notepad++ and published on GitHub. It contains various gifs of monkeys, including some that have become well-known memes on the internet. On top of these gifs, there's an important message to raise awareness about the decline in monkey populations. Towards the end there is a mini game for the audience to have fun with.

XP or 7

2024

Source Code

This project is a journey into nostalgia, blending interactivity with iconic moments from the past. Using tools like p5.fillGradient, SceneManager, and the p5.js Sound Library, I recreated the look and feel of Windows XP and Windows 7 desktops, offering users a chance to revisit these memorable operating systems.

The project begins with a choice: Windows XP or Windows 7. The background for this screen uses a gradient created with p5.fillGradient, adding depth and a polished look. Depending on the user’s selection, they are taken to one of two interactive scenes.

If the user picks Windows XP, they’ll see a desktop that feels like stepping back in time. A notepad opens, and “Dreamscape” by Trance plays in the background—an unmistakable nod to the early YouTube tutorial videos that always seemed to start this way. These videos often used notepad as a way to communicate instructions, which felt simple yet charming.

Choosing Windows 7, however, takes users to a more modern desktop, but with a twist: every time they type something, an error popup appears, complete with the familiar Windows 7 error sound. This scene captures both the nostalgia and the quirks of using these systems, mixing humor with a sense of fond familiarity.

By combining visual effects, sound, and interactivity, this project creates a small but meaningful connection to the past. It’s a way to reflect on how technology shapes our memories, while also celebrating the simple joy of revisiting something familiar.
